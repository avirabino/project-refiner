## 2. Bug Fixes â€” 7 E2E Failures â†’ 0

| # | Symptom | Root Cause | Fix |
|---|---|---|---|
| 1 | ZIP download was 93 bytes | `triggerDownload()` wrapped blob URL *string* in `new Blob()` | Direct anchor-click from blob object URL |
| 2 | No actions in generated spec | `recorder.ts` only sent `ACTION_RECORDED` for SPA navigations; DOM clicks/inputs not dispatched | Added `document.addEventListener('click'/'change')` in `startRecording()`, removed in `stopRecording()` |
| 3 | No `page.goto` for multi-page sessions | Full-page navigation destroys content script; MutationObserver misses it | `recordCrossPageNavigation(sid, document.referrer, location.href)` on content-script reinit |
| 4 | `page.goto` used wrong URL | `action.pageUrl` = fromUrl (source); `action.value` = toUrl (destination) | Changed to `action.value ?? action.pageUrl` in `actionToPlaywright` and `lastNavUrl` tracking |
| 5 | `session.pages[0]` = popup URL | `activeTabUrl` from popup = `chrome-extension://...` when popup is active tab | Filter in `createSession()`: exclude `chrome-extension://` and `chrome://` URLs |
| 6 | `tsc` fails with syntax error on generated spec | `escapeRegex` didn't escape `/`; URL `http://localhost:38470/` embedded as `/http://localhost:38470//` â€” `//` terminated regex then started a line comment (parse error, not fixable by `// @ts-nocheck`) | Added `/` to `escapeRegex` char class: `[.*+?^${}()\|[\]\\\/]` |
| 7 | `tsc` fails with 6 TS2717 errors on transitive types | `rrweb` â†’ `@types/css-font-loading-module@0.0.7` conflicts with TS built-in `lib.dom.d.ts`. Project `skipLibCheck` hides it but test's standalone `tsc` does not | `npm overrides` â†’ local empty stub at `stubs/empty-css-font-types/` |

*Bonus fix:* `tests/fixtures/target-app/about.html` was missing `input-name`/`input-email` fields required by `playwright-export.spec.ts:85`.

## 4. Architecture Decisions

| ADR | Decision | Rationale |
|---|---|---|
| S02-001 | rrweb-player inlined as UMD in replay HTML | Self-contained, offline-safe |
| S02-002 | CSS string selectors in codegen (`page.click(string)`) | Type-valid; satisfies `content.includes('page.click')` QA assertion |
| S02-003 | `// @ts-nocheck` prepended to `.spec.ts` | Prevents module-not-found parse errors in OS temp dir tsc invocation |
| S02-004 | JSZip `generateAsync({ type: 'blob' })` | Client-side ZIP, no server |
| S02-005 | `// BUG:` comments by timestamp comparison | Links test positions to original bug context |
| S02-006 | Keyboard shortcut DOM fallback in content script | `chrome.commands.onCommand` unreachable from Playwright synthetic events |
| S02-007 | `recordCrossPageNavigation()` via `document.referrer` | Captures non-SPA full-page navigations |
| S02-008 | `npm overrides` for `@types/css-font-loading-module` | Resolves rrweb transitive type conflict with TS built-in DOM types; survives `npm install` |
| S02-009 | `escapeRegex` must escape `/` | Forward slashes terminate JS regex literals prematurely |
